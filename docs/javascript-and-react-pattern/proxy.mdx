JS 有內建的 [Proxy](https://developer.mozilla.org/zh-TW/docs/Web/JavaScript/Reference/Global_Objects/Proxy) Object 可以使用，在操作 Object 內的屬性會透過多一層的 Proxy 來進行。

原先是直接取用與更新
