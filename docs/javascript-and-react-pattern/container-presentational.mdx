React Component 的一種設計模式，不過使用 hook 後已經不需要了。

分爲 Container Component 與 Presentational Component，Container 的職責負責獲取資料在由 Presentational Component 進行畫面的呈現。

好處是可以讓關注點進行分離、Presentational Component 易於重覆使用。
