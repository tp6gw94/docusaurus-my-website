在 ES2015 之前 JS 都是採用 implied global，所以在之前的 JS 中沒有 module 的概念，但是在 ES2015 之後，JS 有了 module 的概念，我們可以使用 import 和 export 來實現模組化。

透過 module pattern 可以實現 Encapsulation(封裝) 和 Reusability

:::info INFO

Encapsulation: 值、function 等的 scope 只會在 module 內，外部的 module 皆無法獲取。

:::

:::info INFO
Reusability: 重複使用 module
:::
